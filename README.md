# Workforce Data Analysis: Salary and Remote Work Trends

## Project Overview

This project analyzes workforce salary data using SQL Server to identify compensation trends, remote work adoption, workforce distribution, and geographical salary differences.

The analysis focuses on job titles, experience levels, company sizes, employee locations, and salary growth patterns to generate business insights and support workforce planning.

## Objectives

* Analyze salary trends across countries and job titles
* Compare compensation by experience level
* Study remote work adoption
* Identify high-paying roles and locations
* Support talent acquisition and retention strategies

## Dataset Information

**Dataset:** Data Science Salaries Dataset (2020–2024)

* work_year

* company_size

## Key Findings

* Senior and Executive professionals receive the highest salaries.
* Remote work adoption has increased significantly.
* Large companies employ a substantial share of the workforce.
* Salary levels vary considerably across countries.
* AI and Data-related roles show strong salary growth.

## Business Recommendations

* Implement competitive salary benchmarking.
* Expand recruitment in high-growth regions.
* Increase flexible work opportunities.
* Invest in employee retention and upskilling programs.
